## Open Ai
### GET /openai/menu 

## Description

- Create Menu With Ai Based On User Calory Limit And Extra Restriction



## Request

- Headers 
````json
{
    "access_token": "string"
}
````

````json
    "dinner": "string",

}
````

````json
}
````

````json
{
    "access_token": "string"
}
````
- Body
````json
{
    "eaten": "string|breakfast|lunch|dinner|snack",
    "calorie": "integer"
}
````

_201 - Ok_

````json
{
    "message": "Food has been inputed" 
}
````
